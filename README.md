# OpenClaw (小龙虾) Wiki

OpenClaw 是一款运行在本地的 AI 助手，支持技能管理、自动化工作流、多 Agent 协作、平台集成等功能。

## 📚 目录结构

| 目录 | 说明 |
|------|------|
| `安装配置` | 安装教程、核心概念入门、QMD 本地模型配置 |
| `技能管理` | Skills 技能包的安装、使用与管理 |
| `自动化工作流` | Cron 定时任务、工作流编排 |
| `平台集成` | 微信、企业微信、飞书等平台接入 |
| `实战场景` | 实际应用场景与案例 |
| `故障排查` | 常见问题与解决方案 |

## 🚀 快速开始

1. 下载安装：访问官网获取最新版本
2. 首次启动：登录并完成初始化配置
3. 配置 Skills：安装所需技能包
4. 开始使用：通过自然语言与 AI 交互

## 📖 学习路径

- **入门** → 安装配置 → 核心概念 → 技能管理
- **进阶** → 自动化工作流 → 平台集成 → 多 Agent 协作
- **精通** → 实战场景 → 故障排查 → 自定义开发
